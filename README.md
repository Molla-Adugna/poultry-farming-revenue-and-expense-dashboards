## 📌 Overview
The MFIPF Excel Dashboard is a complete business intelligence solution for poultry farm management. It transforms raw records into actionable insights using interactive dashboards and VBA automation.

This project is officially permitted by the owner of the dashboard to be showcased in this GitHub portfolio for demonstration and professional purposes.

## 📊 Features

| **Feature** | **Description** |
|-------------|-----------------|
| **Data Entry Sheet** | Structured input form for daily farm records (flock size, feed, mortality, production). |
| **Revenue Dashboard** | Visual KPIs showing sales trends, egg/meat revenue, and growth analysis. |
| **Expense Dashboard** | Detailed breakdown of feed, labor, utilities, and other costs with charts. |
| **VBA Automation** | One‑click buttons for report generation, navigation, and data refresh. |
| **Feed Sufficiency Tracker** | Calculates feed consumption vs. stock to prevent shortages. |
| **Usage Guide** | Step‑by‑step documentation for easy setup and operation. |
| **Customization Optionsvigation


## ⚙️ Tech Stack
**- Excel –** PivotTables, charts, conditional formatting

**- Power Query –** ETL automation for cleaning and transforming data

**- VBA Macros –** interactive buttons and workflow automation

## 🚀 Installation & Usage
bash  

```git clone https://github.com/yourusername/MFIPF-Excel-Dashboard.git```

1. Open the Excel file in Microsoft Excel 2016 or later
2. Enable Macros to activate VBA buttons
3. Input records in the Data Entry Sheet
4. Use toggle buttons to switch between dashboards
5. Press Refresh to update with the latest data
